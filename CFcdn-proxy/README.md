## `_workers3.js` 说明
反向代理加速任意网站，带web界面  
[使用教程](https://github.com/ymyuuu/Cloudflare-Workers-Proxy/blob/main/README.md)  
示例请求：`https://your-worker-url.com/https://example.com/`  
将`your-worker-url.com`替换为您的`Cloudflare Workers URL`，`example.com`替换为您要代理的目标网站的地址
